# Notso.fr Design system

## Principles

Based on ITCSS :
- https://www.xfive.co/blog/itcss-scalable-maintainable-css-architecture/

## Libraries

Using the libraries :
- https://feathericons.com/

## Build

### CSS

To compile the css use :

```
npm run build
npm run compress

# or, using yarn,

yarn build
yarn compress
```

To watch the changes :
```
npm run concurrent

# or, using yarn,

yarn concurrent
```

### Documentation

To serve the demonstration pages, use :
```
npm run dev

# or, using yarn,

yarn dev
```

To build only the pages, use :
```
npm run build:pages

# or, using yarn,

yarn build:pages
```
